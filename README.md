# mosip-inji-face-sdk

Inji face sdk for mosip

## Installation

```sh
npm install mosip-inji-face-sdk
```

## Usage

```js
import { multiply } from 'mosip-inji-face-sdk';

// ...

const result = await multiply(3, 7);
```
